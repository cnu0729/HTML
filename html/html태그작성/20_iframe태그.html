iframe 사용하기

            iframe 웹페이지나 웹에 있는 정보를 포함해서 보여주는 태그
            html5 이후부터는 사용을 많이 하지 않음
            html 포함해서 보여주기도 함

            scr="" 불러올 주소 설정 위치 설정
            width="" 가로 길이 지정
            height="" 세로 길이 지정
            frameborder="" 테두리 지정 0으로 설정하면 테두리 없음
        
            iframe 외부에서 태그해서 볼 수 없도록 지정해놓으면
            엑스박스로 되면서 홈페이지에서 연결을 거부했습니다. 낭모
            외부에서 볼 수 있도록 지정해놓으면
            html에서 src 안에 작성한 홈페이지를 볼 수 있음
