이미지 태그 img

        - 웹 문서에 사진이나 그림 같은 이미지를 삽입하는 용도의 태그

        [속성]
        (속성 : 태그에 작성하는 내용)

        1. src    : 삽입할 이미지의 경로를 작성하는 속성
                    (파일 위치, 인터넷 주소 = 이미지를 가져올 수 있는 경로)
        2. width  : 가로 길이를 지정해주는 속성 (필수X)
        3. height : 세로 길이를 지정해주는 속성 (필수X)
        4. alt    : 이미지 설명을 작성하는 속성
                    → 이미지가 보이지 않을 때 화면에 출력되는 그림
                    + 이미지 설명을 읽어주는 역할

인터넷에 존재하는 주소로 이미지 출력하기

이미지 절대 경로로 출력하기

        절대 경로 : 변하지 않는 기준점으로 부터 목표 까징듸 경로
        (서울특별시 강남구 테헤란로 14길 6 남도빌딩)

        - 윈도우 기준으로는 서울특별시가 C드라이브, D드라이브가 기준점

상대 경로로 이미지 출력하기

        상대 경로 : 현재 파일/페이지 위치를 기준으로 하여
                    목표까지의 경로를 나타낸 것
                    (파일 기준에서부터 시작하는 위치)
                    (학원에서 엘리베이터를 타고 1층으로 내려가서 오른쪽에 세븐일레븐이 있음)

                    코드에서 뒤로가기를 작성할 때는 ../ 를 이용해서 뒤로가기 진행한다.
                    cd = change Directory (폴더 변경)
                    뒤로가기를 세번 진행하고 싶다면 ../../../ 뒤로가기 세번 작성 완료

고정 크기 단위(px) : 항상 고정된 크기를 지님
